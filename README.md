# Netflix Data Analysis & Recommendation System

## 📌 Project Overview
This project is an interactive analysis and recommendation system for Netflix content. It demonstrates **data cleaning, exploration, visualization, and content-based recommendations** using Python. The analysis is performed in a **Jupyter Notebook**.

---

## 💡 Key Features
- Cleaned and analyzed **8,000+ Netflix titles** with Python and Pandas
- Visualized insights about:
  - Movies vs TV Shows distribution
  - Top producing countries
  - Popular genres
  - Netflix content growth over time
  - Rating distribution
- Developed a **content-based recommendation system** using:
  - Genres and description features
  - Cosine similarity
